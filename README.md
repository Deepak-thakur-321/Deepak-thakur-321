## ⚡ Who Am I

I'm a **Frontend & MERN Stack Developer** who spent the last 6 months in a professional dev role — shipping real features, fixing real bugs, and learning what it actually takes to build production software.

Now I'm doing something most devs don't: **deliberately pivoting into QA Automation with Playwright** — because I believe the best engineers understand *quality*, not just code.

I don't just write code. I think about **why it breaks**, **how to prevent it**, and **what it means to ship something you can actually trust**.

- 🔭 Currently building: **Production-grade Playwright automation frameworks**
- 🧠 Deep focus on: **E2E testing, component testing, CI integration**
- 🛠️ Background in: **React, Next.js, Node.js, Express, MongoDB**
- 📍 Target: **QA Automation Engineer / SDET role in 2026**
- ⚡ Fun fact: I obsess over UI aesthetics *and* test coverage — rare combo
